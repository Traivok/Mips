Os dois executáveis abaixo fazem a mesma coisa. A diferença é que o arquivo que tem a extensão .jar necessita do java
instalado na máquina para funcionar, enquanto o .exe não necessita do java mas necessita do java instalado e funciona
nos PC's do CIn independente deles terem ou não o JAVA instalado

Link para tutorial de configuração da memória e uso do montador:
https://docs.google.com/document/d/1EwvbpNryvONlVZRe7ENlwCnVsXgHkTS2Al1KCAHGPJ0/edit?usp=sharing
